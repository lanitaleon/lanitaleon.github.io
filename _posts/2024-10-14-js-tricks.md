# 1. 浏览器查看 PDF 体感过亮

F12 在 pdf embed 样式中增加 filter，要黑灰白调比例就可以。

https://developer.mozilla.org/en-US/docs/Web/CSS/filter

```css
element.style {
	filter: invert(8%);
}
````

# 2. 一键回顶部

新建书签如下

```js
javascript:(function(){window.scrollTo(0,0);})();
```
